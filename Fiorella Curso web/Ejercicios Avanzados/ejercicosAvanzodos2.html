Formas de ejecutar un código javascript

Hay dos formas distintas:

Ejecución directa

Las instrucciones se colocan dentro de la etiqueta "script". Cuando el navegador lee la página y encuentra un script va interpretando las líneas de código y las va ejecutando una después de otra.

Respuesta a un evento

Los programas como javascript están preparados para atrapar determinadas acciones realizadas por el usuario llamadas eventos y realizar acciones como respuesta. De este modo se pueden realizar programas interactivos, ya que controlamos los movimientos del usuario y respondemos a ellos.
